# Lumen Prints Website

## Getting Started

To set up the project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/tejasvimys/lumen-prints-website.git
   cd lumen-prints-website
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## Deployment
This project is configured to deploy to GitHub Pages using GitHub Actions. Make sure your repository is configured to use GitHub Pages from the `gh-pages` branch.

## Project Structure

- `pages/`
- `components/`
- `styles/`
