# Lumen Print Website

Welcome to the Lumen Print Website project! This project is designed to showcase a colorful printing company, providing information about our services, gallery, and contact options.

## Project Structure

```
lumen-print-website
├── src
│   ├── index.html          # Main entry point of the website
│   ├── about.html          # Information about the company
│   ├── services.html       # Details of printing services offered
│   ├── gallery.html        # Gallery of completed projects
│   ├── contact.html        # Contact form and information
│   ├── css
│   │   ├── style.css       # Main styles for the website
│   │   └── responsive.css   # Responsive design styles
│   ├── js
│   │   ├── main.js         # Main JavaScript functionality
│   │   └── form-validation.js # Form validation script
│   └── assets
│       └── fonts           # Font files for typography
├── package.json            # npm configuration file
└── README.md               # Project documentation
```

## Features

- **Homepage**: A vibrant landing page that introduces visitors to our printing services.
- **About Us**: Learn about our history, mission, and values.
- **Services**: Detailed descriptions of our printing services, including features and benefits.
- **Gallery**: A showcase of our completed projects to attract potential customers.
- **Contact**: A contact form for inquiries and quotes, along with our contact information.

## Setup Instructions

1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd lumen-print-website
   ```
3. Install dependencies:
   ```
   npm install
   ```
4. Open `src/index.html` in your browser to view the website.

## Usage Guidelines

- Customize the content in the HTML files to reflect your company's information.
- Update styles in `src/css/style.css` and `src/css/responsive.css` to match your branding.
- Use the JavaScript files to add interactivity and enhance user experience.

For any questions or contributions, feel free to reach out!
